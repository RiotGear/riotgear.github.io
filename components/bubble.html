Hover over this sentence to see a tooltip bubble appear

Use

HTML 

<rg-bubble>hover over me</rg-bubble>

JavaScript 

var tags = riot.mount('rg-bubble', {
  bubble: {
    text: 'Bubbles go pop!'
  }
});

The tooltip pops up on hover. Once the mouse leaves, it disappears after 1 second.
