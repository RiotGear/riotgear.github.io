--- layout: page title: Contributing ---

Pull Requests, suggestions or comments about RiotGear are all welcome and valued.

To start contributing to the libraray please follow these steps:

Step 1

Fork the repo!

Step 2

Write some code...

Step 3

Submit your Pull Request so that we can review the code before merging into master.

Sit back

At some point your changes will get merged in and we'll publish a new version of RiotGear!

Suggestions

If you'd like to suggest new components or enhancements to existing ones please raise an issue or talk about them in the forum.

Code Conduct

We're all friends! This project adheres to the Open Code of Conduct. By participating, you are expected to honor this code.
